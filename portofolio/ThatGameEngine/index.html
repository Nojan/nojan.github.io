That Game Engine

ThatCloudEngine is a game engine I wrote that can run assets from ThatCloudGame. I discovered Cloud while I was a teenager, and it became one of my comfort games and an inspiration to my career.

Yet, I wanted to be able to play it on more devices. I realized that most of the assets are in known formats, so I could “just” reimplement the engine. And I added touch control to play it on my phone.

See the code on github: https://github.com/Nojan/ThatCloudEngine
